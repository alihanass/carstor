# 🚗 VelocityDrive – Premium Car Store

**VelocityDrive** is a fully responsive, modern web application that lets users browse a curated collection of high-performance and luxury vehicles, add them to a shopping cart, manage their personal profile, and complete a simulated checkout. The project showcases a clean UI, real-time cart updates, local storage for profile persistence, and dynamic filtering.

## ✨ Features

- **Car Catalog** – Grid layout displaying 10+ premium cars with images, specs, and prices.
- **Dynamic Filtering** – Filter by category: All, Sport, Luxury, Electric, SUV.
- **Real‑time Search** – Search cars by model name instantly.
- **Shopping Cart** – Add/remove items, adjust quantities, see live total and item count.
- **User Profile** – Modal with editable name, email, and bio. Data persists using `localStorage`.
- **Responsive Design** – Works seamlessly on desktop, tablet, and mobile devices.
- **Smooth UI/UX** – Hover effects, sticky navbar, animated cart sidebar, and clean typography.

---

## 🛠️ Built With

- **HTML5** – Semantic structure.
- **CSS3** – Custom styling, Flexbox, Grid, and responsive breakpoints.
- **JavaScript (ES6)** – DOM manipulation, cart logic, filtering, local storage, and modal handling.
- **Font Awesome 6** – Icons for visual enhancement.
- **Google Fonts (Inter)** – Modern sans‑serif typography.
